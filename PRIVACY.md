# TaskPlanner Privacy Policy

Effective date: 2026-07-21

TaskPlanner stores task-board data as markdown and JSON files inside the repository selected by the user. The TaskPlanner extension and skills-only plugin do not require a TaskPlanner account, collect analytics, run advertising, sell personal data, or transmit repository content to a TaskPlanner-operated service.

When TaskPlanner is used through VS Code, Cursor, Codex, ChatGPT, a source-control provider, or another host, that host may process prompts, repository files, diagnostics, or marketplace information under its own privacy policy. TaskPlanner does not control those host services.

Users control and may edit or delete all TaskPlanner project data in `.tasks/`, the managed marker blocks in `AGENTS.md`, `CLAUDE.md`, and `.cursorrules`, and the optional README attribution block. Disabling README attribution prevents future insertion but intentionally does not remove existing text.

Security or privacy questions may be reported through [TaskPlanner GitHub Issues](https://github.com/smekai/taskplanner/issues).
